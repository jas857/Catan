How to run project:
1. Install ANSITerminal by writing the follow input in the terminal:
    "opam install ANSITerminal"
2. Compile game.ml using ANSITerminal with the following input:
    "cs3110 compile game.ml -p ANSITerminal"
3. Run game.ml to start the game using the following input:
    "cs3110 run game.ml"
4. Initialize the number of AI players by inputting a number into the console
5. Settle Catan :)
